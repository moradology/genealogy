# Research Memory

This directory stores durable research memory for the genealogy artifact.

## Layout

- `reasoning-traces/` - structured notes for fruitful reasoning chains that should survive beyond chat history.
- `sources/` - source listings, indexing blurbs, source ids, source-type tags, and retrieval notes.

## When To Add A Reasoning Trace

Add a trace when a line of thought is useful enough that we would want to recover it later, especially when:

- It identifies a specific next record pull.
- It explains why a location or surname variant matters.
- It separates a plausible hypothesis from proved fact.
- It records a conflict, same-name collision, or exclusion clue.
- It connects a map/geography decision to evidence.

## Trace Discipline

Each trace should answer:

- What question were we trying to answer?
- What evidence made the hypothesis plausible?
- What is still not proved?
- Which source ids support the reasoning?
- What exact next move would improve or falsify the claim?

Keep traces concise. They are memory anchors, not essays.

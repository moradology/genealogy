# Reasoning Traces

Reasoning traces preserve useful research paths that are not yet fully settled.

Use `template.md` for new traces. Keep the front matter stable so traces can later be indexed or converted to JSON.

## Status Values

- `active` - current high-value path.
- `parked` - useful but not next in line.
- `resolved` - proved or disproved and incorporated into the main artifact.
- `rejected` - plausible path that was ruled out.

## Confidence Values

- `documented`
- `strong`
- `lead`
- `working`
- `possible`
- `open`

# Ancestry pull sheet (2026-07-08)

One subscription month, one browser session, ~15 lookups. Each pull names its
collection, search terms, what to save, and the Docket case it feeds. Save
every record image (or index detail page where no image exists) plus
Ancestry's own source citation text. Pulls are ordered by tree impact.

## Protocol

Saved files go to a single folder (any name; tell the session where). Each
pull's saved artifacts: the record image, the Ancestry citation text, and the
URL. Cross-verification then runs LOCALLY against the page, the geojson, and
the traces — every field compared, every conflict surfaced through the Docket,
never silently resolved. Nothing lands without the standard re-verification.
